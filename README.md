# AgroInvest 🌱


AgroInvest is a PHP and MySQL based agriculture investment platform where farmers and investors can interact efficiently.

## Features
- User Registration
- Login Authentication
- Farmer Dashboard
- Investor Dashboard
- Investment Management
- Harvest Tracking

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- XAMPP
