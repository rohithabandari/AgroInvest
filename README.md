AgroInvest is a web-based agriculture investment platform that connects farmers and investors.

## Features
- Farmer dashboard
- Investor dashboard
- Investment tracking
- Profit distribution
- Harvest updates
- Secure login system

## Tech Stack
- PHP
- MySQL
- CSS
- XAMPP (Local Server)

## How to Run
1. Install XAMPP
2. Place project inside htdocs
3. Import database
4. Start Apache & MySQL
5. Open localhost/agroinvest
